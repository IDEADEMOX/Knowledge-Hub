# AI Agent 知识库（前端层）

通过 [Lovable](https://lovable.dev/) 实现 AI Agent 知识库的前端层。

UI 需求：

- 登录/注册页面 — 带品牌logo的卡片式表单
- 顶部导航栏 — 登录后显示用户名，支持下拉菜单（个人信息、我的知识库、退出登录）
- 个人信息弹框 — 编辑用户名和邮箱
- 首页 — 居中输入框 + 快捷入口卡片
- 知识库列表 — 表格展示，支持上传（PDF/Word/Excel等）、搜索、删除
- 历史记录 — 对话列表，点击查看详情
- 标签管理 — 创建/编辑/删除标签
- Prompt管理 — 创建/编辑提示词模板

# 产品介绍

一个拥有 AI Agent 管家的私人知识库，帮助用户打理个人知识储备，快速从知识库中检索、整合所需信息。

# 项目启动

```bash
# 安装依赖
npm install

# 启动项目
npm run dev
```
